# webpack-starter

## Quickstart
To set up this project on your local machine:

- Webpack
```
$ npm install webpack -g
```

- Webpack Dev Server
```
$ npm install webpack-dev-server -g
```

- NPM Dependancies
```
$ npm install
```

- Run Local Server
```
$ webpack-dev-server --progress --colors
```

Open http://localhost:8080/webpack-dev-server/bundle
